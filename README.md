# Mina script

=== script.sh ===:
Denna script skriver ut infromation av ditt system så som användarnamn, CPU arkitektur, Privat ip address information, Publik ip address och subnäts mask.

Projekt består ut av bash kod och bör kunna köras i terminalen av systemet, för att köra scriptet se till att det har rätt rättigheter och skriv i terminalen [PATH]/[TO]/
script.sh
